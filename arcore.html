ARCore for any rooted Android device

ARCore is an Augmented Reality SDK developed by google for Android smartphones. ARCore V1.0 is officially released by google. It is supported by only 13 high end android smartphone models.

How to install and run ARCore?

Before you proceed, your device must be rooted, running on Android 7.0 or Higher, need a root file explorer, Buildprop Editor or a text editor.

We are going to modify the build.prop file so that the device can be recognized as a PIxel XL. If camera2 API is not enabled we add a new system property to enable it. Then place a calibration_cad.xml file in the /system/etc/ folder. Calibration_cad.xml file consist camera configurations, that are used by ARCore app.

Tested on Xiaomi Redmi Note 4 MIUI based on Android 7.0 Nougat stock rom and Pixel Experiance Android 8.1 Oreo Custom rom.

Before starting any modifications to system files please take a backup, if any thing goes wrong(though it doesnot), then it will be helpfull.

Use build prop editor app or any text editor for modifying build.prop, by opening it with any root file manager like ES File Explorer or any other app. Modify the build.prop system properties as given below:

  1. ro.build.product
  2. ro.product.device
  3. ro.product.board
  4. ro.product.name

    5. Change above properties values as "marlin", without quotes.

    For example: System Property ro.product.name=Redmi Note 4 as ro.product.name:marlin

  5. ro.product.manufacturer
  6. ro.product.brand
    7. change value to Google
  7. ro.product.model
    8. change value to Pixel XL

    Search for system property persist.camera.HAL3.enabled=1 if it is already there, then you have to skip this step

  8. Add a new System Property:

    persist.camera.HAL3.enabled=1

Now restart the device.

You can check the device name changed to Pixel XL, by opening settings->System->About phone->Model. (You are greeted with a message:"Welcome to PIXEL Family", from google in your gmail)

Now download the file calibration_cad.xml click here to download. Copy the file and paste in the folder /system/etc/

we need to add a reference system property for this file in build.prop. Add a new prooperty in build.prop

ro.config.calibration_cad=/system/etc/calibration_cad.xml

save the file.

Open Play store, search for ARCore app and install.

Thats it. Now your device is ready to run any ARCore app. Run any ARCore app available in playstore or Build a new app using Unity3d. I builded a Hello ARCore click here to download app in unity for testing.

Note: ARCore performs complex computation for tracking and rendering content, performance will be depended on the CPU, memory and GPU. You can replace the original build.prop file if you want, but you will not be able to install other ARCore apps directly from Play Store.
